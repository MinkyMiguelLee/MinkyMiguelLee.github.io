[SpringBoot 스타터 페이지](https://start.spring.io/)

2. 프로젝트 생성

  • **프로젝트 선택**
    - Project : Gradle Project
    - Spring Boot : 2.3.x
    - Language : Java
    - Packaging : Jar
    - Java : 11

  • **프로젝트 메타데이터**
    - Group
    - Artifact - Build 되어 나온 결과물...즉 프로젝트 명

  • **Dependencies**
    - Spring Web, Thymeleaf(Template Engine)
